# 🛒 Consumer Spending Intelligence Platform

A complete data analytics platform built with Python and Streamlit.

## 🔴 Live Demo
👉 [Click here to open the dashboard](https://retail-platform-g4jkqhsvgwicomkvan7agr.streamlit.app)

## 📊 What it does
- Processes 1,067,371 real retail transactions
- Generates £20.4M revenue insights
- Segments 5,861 customers using RFM analysis
- Interactive 4-page dashboard

## 🛠️ Tech Stack
- Python, Pandas, SQLite
- Plotly, Matplotlib
- Streamlit

## 📁 Project Structure
- 01_ingestion.py → loads Excel into database
- 02_cleaning.py → cleans and fixes data
- 03_eda.py → generates business charts
- 04_segmentation.py → RFM customer segments
- app.py → Streamlit dashboard
